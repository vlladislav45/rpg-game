# JRPG Game server

### Back end: 
* Java
* db PostgreSQL


##For contributors/developers

###Quick start:
 ```
// Create a local .env file for basic configuration
cp .env.dist .env

// Start the stack of Docker containers in a terminal
docker-compose up
```
